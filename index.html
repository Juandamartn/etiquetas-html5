<article>

Sirve para especificar contenido independiente. Es soportado por Chrome 6.0 en adelante, internet explorer/edge 9.0 en adelante, Firefox 4.0 en adelante, safari 5.0 en adelante y opera 11.1 en adelante.

Ejemplo

<!DOCTYPE html>

<html>

<body>

<article>

<h1>Ejemplo</h1>

<p>Este es un ejemplo de la etiqueta article</p>

</article>

</body>

</html>

<aside>

Sirve para definir un contenido separado del que se encuentra aunque también se puede poner como una barra lateral en el artículo. Es soportado por Chrome 6.0 en adelante, internet explorer/edge 9.0 en adelante, Firefox 4.0 en adelante, safari 5.0 en adelante y opera 11.1 en adelante.

Ejemplo

<!DOCTYPE html>

<html>

<body>

<p>Todos en algún momento pensamos que los ornitorrincos eran los únicos que nacían en huevo.</p>

<aside>

<h4>Nota </h4>

<p>Existen unos animales que se llaman equidnas que también lo hacen.</p>

</aside>

</body>

</html>

<audio>

Esta etiqueta es para poner elementos de audio en las páginas, pero sólo admite tres tipos de audio, mp3, wav y ogg, aunque internet explorer no acepta el formato wav y ogg y safari no acepta el formato ogg, los navegadores que no soportan la etiqueta audio muestran el texto que está entre la etiqueta. Es soportado por Chrome 4.0 en adelante, internet explorer/edge 9.0 en adelante, Firefox 3.5 en adelante, safari 4.0 en adelante y opera 10.5 en adelante.

Ejemplo

<!DOCTYPE html>

<html>

<body>

<audio controls>

<source src="BUNBURY – Más alto que nosotros sólo el cielo.mp3" type="audio/mpeg">

</audio>

</body>

</html>

<bdi>

Sirve parar aislar una parte de un texto que tiene un formato de escritura diferente a otro texto. Es soportada por Chrome, Firefox, Safari y Opera.

Ejemplo:

<!DOCTYPE html>

<html>

<body>

<ul>

<li>User <bdi< Abu Abdallah Muḥammad ibn Mūsā al-Jwārizmī </bdi>: Matemático</li>

<li>User <bdi> جلال الدين محمد بلخى</bdi>: Poeta</li>

<li>User <bdi> Ruhollah Musaví Jomeini</bdi>: Ayatollah</li>

</ul>

</body>

</html>

<data>

Vincula el contenido con una traducción legible por el ordenador y proporciona ambos valores, el legible por la computadora para procesar datos como el valor legible por los humanos. Es soportada por Chrome 62.0, por Internet Explores/Edge, Firefox 22.0 y por Opera 49.

Ejemplo

<!DOCTYPE html>

<html>

<body>

<ul>

<li><data value="1">Amanita Muscaria</data></li>

<li><data value="69">Shitake</data></li>

<li><data value="16180339887498948">Trufa</data></li>

</ul>

</body>

</html>

<canvas>

La etiqueta <canvas> se utiliza para dibujar gráficos sobre la marcha, a través de secuencias de comandos (generalmente JavaScript). La etiqueta <canvas> es solo un contenedor para gráficos, debe usar un script para dibujar los gráficos. Es soportado por Chrome 4.0, Internet explorer 9.0, Firefox 2.0, Safari 3.1 y Opera 9.0 en adelante.

Ejemplo:

<!DOCTYPE html>

<html>

<body>

<canvas id="myCanvas">Your browser does not support the HTML5 canvas tag.</canvas>

<script>

var c = document.getElementById("myCanvas");

var ctx = c.getContext("2d");

ctx.fillStyle = "#FF0000";

ctx.fillRect(0, 0, 80, 100);

</script>

</body>

</html>
